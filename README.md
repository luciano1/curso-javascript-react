# Curso JavaScript e React

Trilha didática para aprendizagem de JavaScript, componentes, estado e construção de interfaces com React.

## Objetivo

Conduzir o estudante dos fundamentos da linguagem JavaScript até a criação de pequenos projetos com React.

## Conteúdos previstos

- Variáveis, tipos e operadores
- Condicionais e repetições
- Funções
- Arrays e objetos
- DOM e eventos
- LocalStorage
- Componentes React
- Estado e propriedades
- Organização de projetos
- Projetos guiados

## Projetos possíveis

- Calculadora
- Lista de tarefas
- Cadastro simples
- Loja modelo
- Catálogo de produtos
